# ml-reentry-mlir

Prototype ahead-of-time compiler for Graph Neural Networks (PyTorch Geometric →
MLIR sparse-tensor dialect → GPU code). Working CLI/package name: `gnnc`.

See [internal-docs/project-summary.md](internal-docs/project-summary.md) for the
project's goal, design decisions, and sprint structure. Setup status and
day-zero readiness checklist live in
[internal-docs/repo-setup-plan.md](internal-docs/repo-setup-plan.md).

Build/usage docs land once Section 4 (LLVM + torch-mlir submodule builds) of
the setup plan completes.
